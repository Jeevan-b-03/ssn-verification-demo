Compliance & Usage

Follow SSA policies when integrating real services. This demo includes a mock verifier. Replace with SSA services only if eligible and with written consent (SSA‑89).

SSNVS (Social Security Number Verification Service)

  • For wage reporting (Form W‑2) purposes only.
  • Verify up to 10 names/SSNs online or upload up to 250k for next‑business‑day results.

Official page: ssa.gov/employer/ssnv.htm

eCBSV (electronic Consent Based SSN Verification)

  • API returns yes/no match and death indicator (if present).
  • Requires the SSN holder’s written consent (SSA‑89).
  • Limited to Permitted Entities.

Overview: ssa.gov/dataexchange/eCBSV/

SSA‑89 (Written Consent)

  • One‑time use; typically valid for 90 days unless initialed otherwise.
  • Must include name, DOB, SSN, purpose, and signature (wet or electronic meeting requirements).

Form: ssa.gov/forms/ssa-89.pdf · Consent guidance: eCBSV Written Consent

Reminder: Never store plaintext SSNs; keep masked + hashed data and immutable audit logs.
